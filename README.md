# Screen Fill Light

A simple Android fill-light app built with Kotlin and Jetpack Compose.

## Features

- Immersive fullscreen color panel
- Window brightness forced to maximum while the app is active
- Optional keep-screen-awake toggle
- Preset swatches plus RGB custom color sliders
- Recent color history stored with DataStore
- Countdown auto-close options

## Project Notes

- `minSdk` is 26
- The app adjusts only its own window brightness
- No camera or system settings permissions are required

## Open And Run

1. Open the folder in Android Studio.
2. Let Android Studio install the required SDK components.
3. Sync the Gradle project and run on a device.
